# HR Recruiter — Повний дизайн-проєкт для Google Stitch

> HR-рекрутинг з AI-асистентом для автоматизації підбору персоналу. Веб-додаток (React + Vite + shadcn/ui). Функціонал: управління вакансіями, резюме, AI-матчинг, пошук на job boards (Work.ua, Robota.ua, LinkedIn), аналітика зарплат, задачі, щоденні зведення, email-комунікація.
>
> Цільова аудиторія: HR-рекрутери, рекрутингові агенції, small business owners.
